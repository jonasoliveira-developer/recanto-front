# Instruções para IA

- Utilize sempre boas práticas de desenvolvimento.
- Prefira a componetização: separe funcionalidades em componentes reutilizáveis.
- Nunca abrevie nomes de variáveis, funções ou componentes.
- Use nomes completos e significativos para facilitar entendimento e manutenção.
- Sempre escreva e comente o código em português.
- Seja direto: explique pouco e implemente mais.
- Priorize clareza e organização no código.
- Evite comentários desnecessários, mas documente o essencial.
- Siga padrões de projeto e mantenha a estrutura do projeto consistente.
